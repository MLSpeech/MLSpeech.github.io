Access the Deep Phonetics Tools Tutorial here

This tutorial shows the basic command-line usage for each of the tools described on this site. It was originally designed for the BigPhon workshop at LabPhon15. This tutorial steps through each of the tools with a set of sample wav files (stop-initial CVC words recorded by a male and a female speaker) to measure vowel duration, estimate formants, measure word duration, and estimate VOT of the initial stop consonant, for each file. The tutorial first demonstrates the use of each tool to process a single file, and then demonstrates the functionality for processing an entire directory of audio files at once.

The tutorial includes links to all tools, the set of sample wav files, and sample R code that can be used to analyze the output of each tool. The tutorial can be cloned as an entire repository directly onto users' local machines. However, be sure to see each tool's individual page for important information about dependencies and any installation instructions:
